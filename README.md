**📊 Sales Annual Report Dashboard (Excel)**

Welcome to the Sales Annual Report Dashboard repository! This project showcases a dynamic, interactive Excel dashboard that visualizes annual sales performance across key business dimensions such as gender, product category, geography, order status, and sales channels.


---

**🔍 Overview**

This dashboard was created to provide stakeholders with a clear, data-driven understanding of:

Sales trends over the year

Customer demographics and behavior

Top-performing sales channels

Regional sales performance

Category-level sales contribution

Order status distribution


It is fully interactive, allowing users to filter data by Month, Gender, Channel, and Product Category using slicers.


---

**🧩 Features

📅 Monthly Sales Analysis**

Visualized with bar and line charts to track trends in sales amount and number of orders.


**👥 Customer Segmentation**

Gender split (Men vs. Women) and Age group vs Gender insights.


**📦 Category-wise Sales**

Highlights top-selling product categories (e.g., Sets, Western Dress, Sarees).


**🌐 Regional Sales Performance**

Displays sales volumes from the top 10 states.


**🛍️ Sales Channel Insights**

Pie chart visualization of contribution from channels like Amazon, Flipkart, Myntra, etc.


**✅ Order Fulfillment Overview**

Breaks down the percentage of delivered, cancelled, returned, and refunded orders.


**🔄 Interactive Slicers**

For dynamic filtering by Month, Gender, Category, and Channel.

🚀 Getting Started

1. Clone or download the repository.


2. Open the Sales_Annual_Report_Dashboard.xlsx file in Excel.


3. Use the slicers on the left side of the dashboard to interact with the data.


4. Explore insights and modify filters to derive meaningful analysis.
